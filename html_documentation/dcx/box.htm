>> Introduction


The box control is a container control like the panel. It hosts other controls inside and also has the Cell Layout Algorithm available to manage the layout of child controls automatically.

>> Control Styles


These control styles are available when creating a box control.

Control Styles
center   Control text is centered.
right   Control text is right justified.
bottom   Control text is at the bottom of the box.
squared   Box border is a squared rectangle instead of the default round rectangle.
 
General Styles
tabstop   Specifies a control that can receive the keyboard focus when the user presses the TAB key. Pressing the TAB key changes the keyboard focus to the next control with the tabstop style.
group   Specifies the first control of a group of controls. The group consists of this first control and all controls defined after it, up to the next control with the group style. The first control in each group usually has the tabstop style so that the user can move from group to group. The user can subsequently change the keyboard focus from one control in the group to the next control in the group by using the direction keys.
disabled   The control is created disabled.
notheme   Removes XP Themes on controls.

>> /xdid flags


Control commands are input to the control with the /xdid command.

• -c


This command lets you add a child control to a box control.

Command: /xdid -c dialog ID [ID] [CONTROL] [X] [Y] [W] [H] (OPTIONS)

Example: /xdid -c dcx 4 12 richedit 10 10 400 25 multi

Parms :
[ID]Control ID for the DCX Control. (Must be unique for all the controls of the dialog)
[CONTROL]DCX Control type wanted.
  Value :
pbarCreates a Progressbar control.
treeviewCreates a TreeView control.
toolbarCreates a Toolbar control.
statusbarCreates a Statusbar control.
comboexCreates a ComboEx control.
listviewCreates a Listview control.
trackbarCreates a Trackbar control.
colorcomboCreates a ColorCombo control.
buttonCreates a Button control.
richeditCreates a RichEdit control.
ipaddressCreates a IpAddress control.
updownCreates an UpDown control.
webctrlCreates a Web control.
 
lineCreates a Line control.
boxCreates a Box control.
radioCreates a Radio control.
checkCreates a Check control.
editCreates a Edit control.
scrollCreates a Scroll control.
imageCreates a Image control.
listCreates a List control.
linkCreates a Link control.
textCreates a Text control.
 
dividerCreates a Divider control.
panelCreates a Panel control.
tabCreates a Tab control.
 
windowDocks a mIRC @window. The first parameter in the (OPTIONS) must be the @window name.
dialogDocks a mIRC dialog. The first parameter in the (OPTIONS) must be the dialog name.

[X]X position of control.
[Y]Y position of control.
[W]Width of control.
[H]Height of control.
(OPTIONS)Optional styles and options available on each individual controls.

• -d


This command lets you delete a control added on the box.

Command: /xdid -d dialog ID [CID]

Example: /xdid -d dcx 4 6

• -l


This command lets you add Cell Layout Algorithm rules to your box control for automatic resizing of the child controls.

Command: /xdid -l dialog ID [COMMAND] [PATH] [TAB] [+FLAGS] [ID] [WEIGHT] [W] [H]

Example: /xdid -l dcx 4 root $chr(9) +pv 0 1 0 0

Example: /xdid -l dcx 4 root 1 2 $chr(9) +fi 7 1 200 300

Example: /xdid -l dcx 4 space 1 $chr(9) + 5 5 5 5

Parms :
[COMMAND]Layout command.
  Value :
rootSets the root cell element.
cellAdds a cell element as a child of another cell element.
spaceSets the space padding around a cell element.
updateUpdates the cell layout (must be used after adding rules or changing padding settings to refresh display).

[PATH]Cell Path from root element (can be root to point to root element). See the Cell Layout Algorithm explanation for concrete examples.
[+FLAGS]DCX Control type wanted.
  Value :
fCreates a fixed cell element.
hCreates a width fixed cell if used with f or a horizontal cell pane if used with p.
iIndicates that the supplied control [ID] is valid and is the child control when used with f or l.
lCreates a fill cell element.
pCreates a pane cell element.
vCreates a height fixed cell if used with f or a vertical cell pane if used with p.
wWhen used, it means that the supplied [W] and [H] represent the fixed cell width in the applicable direction (if the fixed cell is fixed in width, height or both).

[ID]ID of the control (used with fixed or fill cells - use 0 if no control si to be linked to the cell).
[WEIGHT]Cell child weight. (used when adding a child cell to a pane cell)
[W]Fixed width of control (used with fixed cell).
[H]Fixed height of control. (used with fixed cell)
(OPTIONS)Optional styles and options available on each individual controls.

N.B. If you use the update command in the init event of a dialog, you will have to use ".timer 0 1 xdid -l dialog update" as a glitch with XP and themes disabled prevents windows from appearing. The timer corrects this problem.

N.B. When using the space command, the [ID] [WEIGHT] [W] [H] are in fact the [LEFT] [TOP] [RIGHT] [BOTTOM] spacings

N.B. Use the value 0 for the fields not used as the command expects them to be filled even though they aren't used.

• -t


This command lets you set the group box text.

Command: /xdid -t dialog ID (Text)

Example: /xdid -t dcx 4 Group Box Text

>> $xdid() Properties


The following properties are used to retreive information in the group box control.

• text


This property lets you retreive the group box text.

Identifier: $xdid(dialog, ID).text

Example: //echo -a $xdid(dcx, 4).text

• inbox


This property lets you retreive interior frame box size that clears the box border and text.

Identifier: $xdid(dialog, ID).inbox

Example: //echo -a $xdid(dcx, 4).inbox

N.B. The return value is X Y W H.

>> Control Events


The following events are fired when a user does some particular actions on the box control. All events and data are sent to the callback alias associated to the dialog when marked. Remember that the events over a child control will be fired according to that child control.

help   Launched when you click on a control using the ? contexthelp button.

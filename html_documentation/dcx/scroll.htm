>> Introduction


This control enables you to create a scroll bar.


>> Control Styles


These control styles are available when creating a scroll bar control.

Control Styles
vertical   Vertical scroll bar.
 
General Styles
tabstop   Specifies a control that can receive the keyboard focus when the user presses the TAB key. Pressing the TAB key changes the keyboard focus to the next control with the tabstop style.
group   Specifies the first control of a group of controls. The group consists of this first control and all controls defined after it, up to the next control with the group style. The first control in each group usually has the tabstop style so that the user can move from group to group. The user can subsequently change the keyboard focus from one control in the group to the next control in the group by using the direction keys.
disabled   The control is created disabled.
notheme   Removes XP Themes on controls.

>> /xdid flags


Control commands are input to the control with the /xdid command.

• -l


This command lets you set the line step value (see Lineup/Linedown events).

Command: /xdid -l dialog ID [VALUE]

Example: /xdid -l dcx 4 5

• -m


This command lets you set the page step value (see Pageup/Pagedown events).

Command: /xdid -m dialog ID [VALUE]

Example: /xdid -m dcx 4 10

• -r


This command lets you set the scrollbar range.

Command: /xdid -r dialog ID [MIN] [MAX]

Example: /xdid -r dcx 4 0 100

• -v


This command lets you set the tracker position.

Command: /xdid -v dialog ID [POS]

Example: /xdid -v dcx 4 50

>> $xdid() Properties


The following properties are used to retreive information in the scroll bar control.

• value


This property lets you retreive the scroll bar position value.

Identifier: $xdid(dialog, ID).value

Example: //echo -a $xdid(dcx, 4).value

• range


This property lets you retreive the scroll bar range.

Identifier: $xdid(dialog, ID).range

Example: //echo -a $xdid(dcx, 4).range

• line


This property lets you retreive the scroll bar line step value.

Identifier: $xdid(dialog, ID).line

Example: //echo -a $xdid(dcx, 4).line

• page


This property lets you retreive the scroll bar page step value.

Identifier: $xdid(dialog, ID).page

Example: //echo -a $xdid(dcx, 4).page

>> Control Events


The following events are fired when a user does some particular actions on the scrollbar. All events and data are sent to the callback alias associated to the dialog when marked.

trackend   When the user releases the left mouse button after dragging the thumb.
 
[VALUE][POS]
[POS]Scroll bar position.
top   When the user presses the home key.
 
[VALUE][POS]
[POS]Scroll bar position.
bottom   When the user presses the end key.
 
[VALUE][POS]
[POS]Scroll bar position.
lineup   When the user presses up/left arrow keys.
 
[VALUE][POS]
[POS]Scroll bar position.
linedown   When the user presses down/right arrow keys.
 
[VALUE][POS]
[POS]Scroll bar position.
pageup   When the user presses the pageup key or clicks on the left/top of the scrollbar thumb track.
 
[VALUE][POS]
[POS]Scroll bar position.
pagedown   When the user presses the pagedown key or clicks on the right/bottom of the scrollbar thumb track.
 
[VALUE][POS]
[POS]Scroll bar position.
 
help   Launched when you click on a control using the ? contexthelp button.

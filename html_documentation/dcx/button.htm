>> Introduction


This control enables you to create a button control.

>> Control Styles


These control styles are available when creating a button bar control.

Control Styles
bitmap   Button is drawed using supplied bitmaps.
default   Button has default focus.
 
General Styles
tabstop   Specifies a control that can receive the keyboard focus when the user presses the TAB key. Pressing the TAB key changes the keyboard focus to the next control with the tabstop style.
group   Specifies the first control of a group of controls. The group consists of this first control and all controls defined after it, up to the next control with the group style. The first control in each group usually has the tabstop style so that the user can move from group to group. The user can subsequently change the keyboard focus from one control in the group to the next control in the group by using the direction keys.
disabled   The control is created disabled.
notheme   Removes XP Themes on controls.

>> /xdid flags


Control commands are input to the control with the /xdid command.

• -k


This command lets you change a caption color or image for the button.

Command: /xdid -k dialog ID [+FLAGS] [COLOR] (FILENAME)

Example: /xdid -k dcx 4 +nd $rgb(255,0,255) C:\mIRC\start.bmp

Parms :
[+FLAGS]Button Color/Image flags
  Value :
dDisabled state color/image.
hHovering state color/image.
nNormal state color/image.
sSelected state color/image.
[COLOR]Caption text color. (Transparency color of the bitmap if the button has style bitmap)
(FILENAME)Bitmap filename. (Only required if style bitmap is used)

• -l


This command lets you set the icon size of the button and clear the image list. SIZE can be 16, 24 or 32. (Not used for bitmap button)

Command: /xdid -l dialog ID [SIZE]

Example: /xdid -l dcx 4 32

• -t


This command lets you set the button caption text. (Not used for bitmap button)

Command: /xdid -t dialog ID (Text)

Example: /xdid -t dcx 4 Button text

• -w


This command lets you add an icon to the button image list.

Command: /xdid -w dialog ID [INDEX] [FILENAME]

Example: /xdid -w dcx 4 113 C:/mIRC/shell.dll

Parms :
[INDEX]Icon index in icon archive (use 0 if the file is a single icon file)
[FILENAME]Icon archive filename

N.B. The insert order is important and is as follows normal, hover, selected and disabled.

>> $xdid() Properties


The following properties are used to retreive information in the button control.

• .text


This property lets you retreive the button caption text.

Identifier: $xdid( dialog, ID ).text

Example: //echo -a $xdid( dcx, 4 ).text

>> Control Events


The following events are fired when a user does some particular actions on the button control. All events and data are sent to the callback alias associated to the dialog when marked. Remember that the events over a child control will be fired according to that child control.
sclick   When you left-click on the button.
 
rclick   When you right-click on the button.
 
help   Launched when you click on a control using the ? contexthelp button.

>> Introduction


This control enables you to create a rebar control. This control acts as a container for other controls. The rebar control is consisted of bands that host an icon, text and a child control.

>> Control Styles


These control styles are available when creating a rebar control.

Control Styles
borders   Borders appear between rebar bands.
varheight   The rebar control displays bands at the minimum required height, when possible. Without this style, the rebar control displays all bands at the same height, using the height of the tallest visible band to determine the height of other bands. Even if this style is applied, bands on the same row will display the height of the highest band on that row.
dblclktoggle   Double click on band text toggles min/max resizing of the band. (Single click is default)
fixedorder   Bands are always in the same order but can be dragged on different lins/columns.
tooltips   The rebar bands have tooltips.
verticalgrip   The size grip will be displayed vertically instead of horizontally in a vertical rebar control. (must be used with vertical)
vertical   The rebar is auto-positioned to the left side of the dialog (gives a vertical rebar).
bottom   The rebar is auto-positioned at bottom of dialog window.
right   The rebar is auto-positioned to the right side of the dialog (must be used with vertical).
noresize   The rebar isn't auto-resized when dialog's size changes.
noparentalign   The rebar isn't auto-positioned when dialog's size changes.
 
General Styles
tabstop   Specifies a control that can receive the keyboard focus when the user presses the TAB key. Pressing the TAB key changes the keyboard focus to the next control with the tabstop style.
group   Specifies the first control of a group of controls. The group consists of this first control and all controls defined after it, up to the next control with the group style. The first control in each group usually has the tabstop style so that the user can move from group to group. The user can subsequently change the keyboard focus from one control in the group to the next control in the group by using the direction keys.
disabled   The control is created disabled.
notheme   Removes XP Themes on controls.

>> /xdid flags


Control commands are input to the control with the /xdid command.

• -a


This command lets you add bands to the rebar. These bands can optionally host a supported child control.

Command: /xdid -a dialog ID [N] [+FLAGS] [CX] [CY] [WIDTH] [ICON] [COLOR] (Item Text)[TAB][ID] [CONTROL] [X] [Y] [W] [H] (OPTIONS)[TAB](Tooltip)

Example: /xdid -a dcx 4 0 +f 0 25 20 2 ComboEx $chr(9) 12 comboex 10 10 400 25 dropedit $chr(9) Tooltip

Parms :
[N]Band position index number (Use 0 to insert band at the end)
[+FLAGS]Item flags
  Value :
cThe button text is colored ([COLOR] is the text color). (Only works with XP themes disabled)
eThe band has an edge at the top and bottom of the child window.
fThe band can't be sized. With this style, the sizing grip is not displayed on the band.
gThe band will always have a sizing grip, even if it is the only band in the rebar.
hThe band will not be visible.
nThe band will never have a sizing grip, even if there is more than one band in the rebar.
wThe band is on a new line.

[CX]Minimal width to which the rebar band child control can be resized. (If you are using the vertical style this value is the minimal height)
[CY]Minimal height to which the rebar band child control can be resized. (If you are using the vertical style this value is the minimal width)
[WIDTH]Width of the rebar band when created . (If you are using the vertical style this value is the height)
[ICON]Icon index number from the icon list (0 for no icon).
[COLOR]Rebar band text color (Only works with XP themes disabled).
(Item Text)Band text (optional).
 
Control Info :
[ID]Control ID for the DCX Control. (Must be unique for all the controls of the dialog)
[CONTROL]DCX Control type wanted.
  Value :
pbarCreates a Progressbar control.
treeviewCreates a TreeView control.
toolbarCreates a Toolbar control.
statusbarCreates a Statusbar control.
comboexCreates a ComboEx control.
listviewCreates a Listview control.
trackbarCreates a Trackbar control.
colorcomboCreates a ColorCombo control.
buttonCreates a Button control.
richeditCreates a RichEdit control.
 
dividerCreates a Divider control.
panelCreates a Panel control.
tabCreates a Tab control.

[X]X position of control. (Needed but ignored by rebar, you can use 0)
[Y]Y position of control. (Needed but ignored by rebar, you can use 0)
[W]Width of control.
[H]Height of control.
(OPTIONS)Optional styles and options available on each individual controls.
 
(Tooltip)Band tooltip (Not functional)

N.B. If the band is the last one created on a band row, the [WIDTH] value is ignored and the band stretches to the end of the rebar control.
N.B. The control part of the command is optional and can be empty if you do not wish to have a control in the band

• -d


This command lets you delete the Nth rebar band.

Command: /xdid -d dialog ID [N]

Example: /xdid -d dcx 4 2

• -i


This command lets you hide the Nth rebar band.

Command: /xdid -i dialog ID [N]

Example: /xdid -i dcx 4 2

• -j


This command lets you show the Nth rebar band.

Command: /xdid -j dialog ID [N]

Example: /xdid -j dcx 4 2

• -k


This command lets you change the icon on a rebar band.

Command: /xdid -k dialog ID [N] [ICON]

Example: /xdid -k dcx 4 2 0

Parms :
[N]Band position index number.
[ICON]Icon index number from the icon list (0 for no icon).

• -l


This command lets you lock the Nth rebar band or all the bands.

Command: /xdid -l dialog ID [N|all]

Example: /xdid -l dcx 4 all

• -m


This command lets you maximize the Nth rebar band.

Command: /xdid -m dialog ID [N]

Example: /xdid -m dcx 4 2

• -n


This command lets you minimize the Nth rebar band.

Command: /xdid -n dialog ID [N]

Example: /xdid -n dcx 4 2

• -t


This command lets you change the text on a rebar band.

Command: /xdid -t dialog ID [N] (ItemText)

Example: /xdid -t dcx 4 2 new text

N.B. Omit text if you want to remove text.

• -u


This command lets you unlock the Nth rebar band or all the bands.

Command: /xdid -u dialog ID [N|all]

Example: /xdid -u dcx 4 all

• -v


This command lets you move the Nth rebar band to the Mth position.

Command: /xdid -v dialog ID [N] [M]

Example: /xdid -v dcx 4 3 1

• -w


This command lets you add an icon to the rebar image list.

Command: /xdid -w dialog ID [INDEX] [FILENAME]

Example: /xdid -w dcx 4 113 C:/mIRC/shell.dll

Parms :
[INDEX]Icon index in icon archive (use 0 if the file is a single icon file)
[FILENAME]Icon archive filename

• -y


This command lets you clear the rebar image list.

Command: /xdid -y dialog ID

Example: /xdid -y dcx 4

>> $xdid() Properties


The following properties are used to retreive information in the listview control.

• text


This property lets you retreive a Nth rebar band text.

Identifier: $xdid(dialog, ID, N).text

Example: //echo -a $xdid(dcx, 4, 1).text

• num


This property lets you retreive the number of bands of a rebar.

Identifier: $xdid(dialog, ID).num

Example: //echo -a $xdid(dcx, 4).num

• .childid


This property lets you retreive Nth rebar band child control ID ($null if no child control on the band).

Identifier: $xdid(dialog, ID, N).childid

Example: //echo -a $xdid(dcx, 4, 2).childid

>> Control Events


The following events are fired when a user does some particular actions on the rebar control. All events and data are sent to the callback alias associated to the dialog when marked. Remember that the events over a child control will be fired according to that child control.

sclick   When a sclick event is fired over a rebar band (excluding the area occupied by a child control).
 
[VALUE][BAND]
[BAND]Rebar band item index number over which the event was triggered.
rclick   When a rclick event is fired over a rebar band (excluding the area occupied by a child control).
 
[VALUE][BAND]
[BAND]Rebar band item index number over which the event was triggered.
change   When the rebar control's height changes because of band movement.
 
[VALUE][W] [H]
[W]New rebar control width.
[H]New rebar control height.
 
help   Launched when you click on a control using the ? contexthelp button.

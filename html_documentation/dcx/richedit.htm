>> Introduction


This control enables you to create a richedit. The richedit interprets the mIRC control codes to display the colors and bold/underline effects. The richedit control loads it's internal Color Palette as the actuel mIRC colors as default behavior. You can modify the colors using the /xdid commands.

The Color Palette is the richedit's internal color list of 16 values used to process mIRC color codes and convert them to disaplyed colors.

>> Control Styles


These control styles are available when creating a richedit control.

Control Styles
multi   Enables a multiline richedit.
center   Text is centered.
readonly   Text cannot be edited manually.
right   Text is right justified.
autohs   Text scrolling is automatically scrolled horizontally without need of a horizontal scrollbar.
autovs   Text scrolling is automatically scrolled vertically without need of a vertical scrollbar.
hsbar   Displays a horizontal scrollbar.
vsbar   Displays a vertical scrollbar.
 
General Styles
tabstop   Specifies a control that can receive the keyboard focus when the user presses the TAB key. Pressing the TAB key changes the keyboard focus to the next control with the tabstop style.
group   Specifies the first control of a group of controls. The group consists of this first control and all controls defined after it, up to the next control with the group style. The first control in each group usually has the tabstop style so that the user can move from group to group. The user can subsequently change the keyboard focus from one control in the group to the next control in the group by using the direction keys.
disabled   The control is created disabled.
notheme   Removes XP Themes on controls.

>> /xdid flags


Control commands are input to the control with the /xdid command.

• -a


This command lets you add text to the end of the richedit.

Command: /xdid -a dialog ID [TEXT]

Example: /xdid -a dcx 4 New Text

• -c


This command will copy the text in the edit to the clipboard.

Command: /xdid -c dialog ID

Example: /xdid -c dcx 4

• -d


This command lets you delete the Nth line of text in the richedit (Only works with multi).

Command: /xdid -d dialog ID [N]

Example: /xdid -d dcx 4 5

• -i


This command lets you insert a line at the Nth position in the richedit. (If the multi style is not used, will overwrite the whole text)

Command: /xdid -i dialog ID [N] [TEXT]

Example: /xdid -i dcx 4 5 New Text

• -k


This command lets you change the richedit background color. (use -1 to revert to default)

Command: /xdid -k dialog ID [COLOR]

Example: /xdid -k dcx 4 $rgb(255,0,0)

• -l


This command lets you change a color in the richedit palette.

Command: /xdid -l dialog ID [N] [COLOR]

Example: /xdid -l dcx 4 5 $rgb(255,0,255)

Parms :
[N]Palette color number 1-16.
[COLOR]RGB Color.

• -m


This command lets you load the mIRC color palette into the richedit.

Command: /xdid -m dialog

Example: /xdid -m dcx 4

• -o


This command lets you overwrite the Nth line in the richedit. (If the multi style is not used, will overwrite the whole text)

Command: /xdid -o dialog ID [N] [TEXT]

Example: /xdid -o dcx 4 5 New Text

• -q


This command lets you load a custom palette into the richedit control.

Command: /xdid -q dialog ID [COLOR1] ... [COLOR16]

Example: /xdid -q dcx 4 $rgb(255,0,0) $rgb(0,0,0)

N.B. The number of colors is limited to 16, everything past the 16th color is ignored.
N.B. You can give the command a number of colors inferior to 16, it will then only add thoses colors in order to the palette starting at 1.

• -r


This command lets you clear the contents of the richedit.

Command: /xdid -r dialog ID

Example: /xdid -r dcx 4

• -t


This command lets you load the contents of a file directly in the richedit.

Command: /xdid -t dialog ID [FILENAME]

Example: /xdid -t dcx 4 C:\mIRC\blah.txt

• -u


This command lets you save the contents of the richedit directly in a file.

Command: /xdid -u dialog ID [FILENAME]

Example: /xdid -u dcx 4 C:\mIRC\blah.txt

• -S


This command lets you set the selection range in the richedit.

Command: /xdid -C dialog ID [START] [END]

Example: /xdid -C dcx 4 1 3

• -Z


This command lets you zoom the text size of the richedit. The percentage of zooming is calculated by dividing the numerator by denominator (NUMERATOR / DENOMINATOR)

Command: /xdid -Z dialog ID [NUMERATOR] [DENOMINATOR]

Example: /xdid -Z dcx 4 200 100

>> $xdid() Properties


The following properties are used to retreive information in the richedit control.

• text


This property lets you retreive the Nth line in the richedit or the whole richedit text.

Identifier: $xdid(dialog, ID, N).text

Example: //echo -a $xdid(dcx, 4, 1).text

N.B. If the multi is not used, you can use $xdid(dialog, ID).text to retrieve the richedit text.

• num


This property lets you retreive the number of lines in the richedit.

Identifier: $xdid(dialog, ID).num

Example: //echo -a $xdid(dcx, 4).num

N.B. Only works with the multi style.

• sel


This returns the index of the first and last selected character position in the control.

Identifier: $xdid(dialog, ID).sel

Example: //echo -a $xdid(dcx, 4).sel

• selstart


This returns the index of the first selected character position in the control.

Identifier: $xdid(dialog, ID).selstart

Example: //echo -a $xdid(dcx, 4).selstart

• selend


This returns the index of the last selected character position in the control.

Identifier: $xdid(dialog, ID).selend

Example: //echo -a $xdid(dcx, 4).selend

• seltext


This returns the selected text in the control.

Identifier: $xdid(dialog, ID).seltext

Example: //echo -a $xdid(dcx, 4).seltext


>> Control Events


The following events are fired when a user does some particular actions on the richedit control. All events and data are sent to the callback alias associated to the dialog when marked. Remember that the events over a child control will be fired according to that child control.

return   When the Enter key is pressed in the richedit control.
rclick   When the right mouse button is pressed on the richedit control.
keydown   When a key is pressed in the richedit control.
keyup   When a key is unpressed in the richedit control.
help   Launched when you click on a control using the ? contexthelp button.

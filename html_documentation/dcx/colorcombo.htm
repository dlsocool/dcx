>> Introduction


This control enables you to create a colorcombo.

>> Control Styles


These control styles are available when creating a colorcombo control.

Control Styles
 
General Styles
tabstop   Specifies a control that can receive the keyboard focus when the user presses the TAB key. Pressing the TAB key changes the keyboard focus to the next control with the tabstop style.
group   Specifies the first control of a group of controls. The group consists of this first control and all controls defined after it, up to the next control with the group style. The first control in each group usually has the tabstop style so that the user can move from group to group. The user can subsequently change the keyboard focus from one control in the group to the next control in the group by using the direction keys.
disabled   The control is created disabled.
notheme   Removes XP Themes on controls.

>> /xdid flags


Control commands are input to the control with the /xdid command.

• -a


This command lets you add a color to the colorcombo at the specified position. If N is 0, it inserts the color at the end.

Command: /xdid -a dialog ID [N] [RGB]

Example: /xdid -a dcx 4 2 $rgb(0,0,255)

• -c


This command lets you select the Nth item in the colorcombo.

Command: /xdid -c dialog ID [N]

Example: /xdid -c dcx 4 2

• -d


This command lets you delete the Nth item in the colorcombo.

Command: /xdid -d dialog ID [N]

Example: /xdid -d dcx 4 2

• -m


This command lets you load all the mIRC colors into the colorcombo.

Command: /xdid -m dialog ID

Example: /xdid -m dcx 4

• -o


This command lets you overwrite the Nth item in the colorcombo.

Command: /xdid -o dialog ID [N] [RGB]

Example: /xdid -o dcx 4 2 $rgb(0,255,0)

• -r


This command lets you clear the contents of the colorcombo.

Command: /xdid -r dialog ID

Example: /xdid -r dcx 4

>> $xdid() Properties


The following properties are used to retreive information in the listview control.

• num


This property lets you retreive the number of items in the colorcombo.

Identifier: $xdid(dialog, ID).num

Example: //echo -a $xdid(dcx, 4).num

• color


This property lets you retreive the Nth colorcombo color.

Identifier: $xdid(dialog, ID, N).color

Example: //echo -a $xdid(dcx, 4, 1).color

• sel


This property lets you the selected item number in the colorcombo.

Identifier: $xdid(dialog, ID).sel

Example: //echo -a $xdid(dcx, 4).sel

>> Control Events


The following events are fired when a user does some particular actions on the colorcombo control. All events and data are sent to the callback alias associated to the dialog when marked. Remember that the events over a child control will be fired according to that child control.

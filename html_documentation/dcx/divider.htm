>> Introduction


This control enables you to create a divider control. This control acts as a container for other controls. The divider control is consisted of two panes that host a child control. The panes are resized using the divider bar between them. The divider control resizes automaticaly it's child windows when the bar is moved. Multiple dividers can be nested for advanced UI interfaces.

>> Control Styles


These control styles are available when creating a divider control.

Control Styles
vertical   The divider has vertical panes.
 
General Styles
tabstop   Specifies a control that can receive the keyboard focus when the user presses the TAB key. Pressing the TAB key changes the keyboard focus to the next control with the tabstop style.
group   Specifies the first control of a group of controls. The group consists of this first control and all controls defined after it, up to the next control with the group style. The first control in each group usually has the tabstop style so that the user can move from group to group. The user can subsequently change the keyboard focus from one control in the group to the next control in the group by using the direction keys.
disabled   The control is created disabled.
notheme   Removes XP Themes on controls.

>> /xdid flags


Control commands are input to the control with the /xdid command.

• -l|r


This command lets you add a child control to a divider pane. The -l switch is for the left/top pane and -r for the right/bottom pane.

Command: /xdid -[l|r] dialog ID [MIN] [IDEAL][TAB][ID] [CONTROL] [X] [Y] [W] [H] (OPTIONS)

Example: /xdid -r dcx 4 100 0 $chr(9) 12 richedit 10 10 400 25 multi

Parms :
[MIN]Minimum Pane Width/Height.
[IDEAL]Extra Not Used Yet.
 
Control Info :
[ID]Control ID for the DCX Control. (Must be unique for all the controls of the dialog)
[CONTROL]DCX Control type wanted.
  Value :
treeviewCreates a TreeView control.
listviewCreates a Listview control.
rebarCreates a Rebar control.
richeditCreates a RichEdit control.
webctrlCreates a Web control.
 
editCreates a Edit control.
imageCreates a Image control.
listCreates a List control.
 
dividerCreates a Divider control.
panelCreates a Panel control.
tabCreates a Tab control.

[X]X position of control. (Needed but ignored by divider, you can use 0)
[Y]Y position of control. (Needed but ignored by divider, you can use 0)
[W]Width of control.
[H]Height of control.
(OPTIONS)Optional styles and options available on each individual controls.

• -v


This command lets you set the divider bar position.

Command: /xdid -v dialog ID [POS]

Example: /xdid -v dcx 4 150

N.B. The value must be between the left pane minimum width/height and the width/height of the control minus the right pane minimim width/height. Left W/H <= POS <= Right W/H

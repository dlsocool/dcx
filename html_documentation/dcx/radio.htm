>> Control Styles


These control styles are available when creating a radio control.

Control Styles
right   Radio control box is at the right of text.
center   Control text is centered.
rjustify   Control text is right justified.
pushlike   Makes the radio act and look like a push button.
 
General Styles
tabstop   Specifies a control that can receive the keyboard focus when the user presses the TAB key. Pressing the TAB key changes the keyboard focus to the next control with the tabstop style.
group   Specifies the first control of a group of controls. The group consists of this first control and all controls defined after it, up to the next control with the group style. The first control in each group usually has the tabstop style so that the user can move from group to group. The user can subsequently change the keyboard focus from one control in the group to the next control in the group by using the direction keys.
disabled   The control is created disabled.
notheme   Removes XP Themes on controls.

>> /xdid flags


Control commands are input to the control with the /xdid command.

• -c


This command lets you check the radio control.

Command: /xdid -c dialog ID

Example: /xdid -c dcx 4

• -t


This command lets you set the radio text.

Command: /xdid -t dialog ID (Text)

Example: /xdid -t dcx 4 Radio Text

• -u


This command lets you uncheck the radio control.

Command: /xdid -u dialog ID

Example: /xdid -u dcx 4

>> $xdid() Properties


The following properties are used to retreive information in the radio control.

• text


This property lets you retreive the radio text.

Identifier: $xdid(dialog, ID).text

Example: //echo -a $xdid(dcx, 4).text

• .state


This property lets you retreive radio selection state.

Identifier: $xdid(dialog, ID).state

Example: //echo -a $xdid(dcx, 4).state

>> Control Events


The following events are fired when a user does some particular actions on the radio control. All events and data are sent to the callback alias associated to the dialog when marked. Remember that the events over a child control will be fired according to that child control.

sclick   When the radio is selected.
 
rclick   When you right-click on the radio.
 
help   Launched when you click on a control using the ? contexthelp button.

>> Introductie


Deze control stelt je in staat om een @window in te voegen als een control in een dialog. De @window moet worden geopened voor dat deze kan worden ingevoegd in een dialog met deze control.

Je kan zoveel @windows toevoegen als je wilt mits deze op zich zelf staan.

Hou alstublieft in de gaten dat als de dialog word gesloten, de @windows worden vrijgegeven. Deze moeten dus nog gesloten worden.
